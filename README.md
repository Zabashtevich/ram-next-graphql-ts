SSR приложение на базе RAM API, построенного по вселенной мультика Рик и морти. 

Особенности:

1. Запросы на сервер реализуются при помощи GraphQL
2. Unit testing
3. Typescript
4. SSR при помощи Next JS
5. Styled components
